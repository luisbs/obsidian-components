## CodeBlock with annotation

### YAML

```use book
title: 'Lord of the rings'
```

### JSON

```use book
{
  "title": "Lord of the rings"
}
```

## CodeBlock with escaped param

### YAML

```use
__file: book
title: 'Lord of the rings'
```

### JSON

```use
{
  __file: 'book',
  title: 'Lord of the rings',
}
```
